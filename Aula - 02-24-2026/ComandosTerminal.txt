Compactar Arquivos

* INFORMATICA RAIZ
   cmd - PROMPT DE COMANDO
   tree - Arvóre de Estrutura do windows
   dir /w /s - Efeito Matriz
   dir-LISTA DIRETÓRIOS
   mkdir - CRIA DIRETÓRIOS
   cd + Nome do diretorio- ENTRA DIRETÓRIOS
   cd.. VOLTA DIRETÓRIOS
   type nul > nomedoarquivo.txt - CRIA O ARQUIVO
   copy con nomedoarquivo.txt - cria um arquivo com um texto dentro use enter e ctrl + C para gravar
   del- apaga arquivos
   rd + nome do diretorio - apaga diretório
   rename "teste.txt" "alter.txt" - renomear arquivo
   systeminfo- mostra a configuração do computador
   cls - limpa a tela
